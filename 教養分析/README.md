# 教養分析

TEPS 第二波「主要照顧者教養方式 × 學業成就」分析的程式碼與報告。重現步驟、環境需求與
資料取得方式見 repo 根目錄的 `README.md`；本檔補充方法細節。

## 方法摘要

1. **品質過濾**：`w2refuse` / `w2sumerr` / `w2sumlog` 三個品質旗標皆為 0
   （13,247 → 7,464）。
2. **主要照顧者**：逐角色取「同住 ∪ 照顧」聯集（`w2s456x` ∪ `w2s457x`，問的是國小
   畢業前）。排除繼父母進聯集者、無親屬者 → **最終 7,280 人**。
3. **照顧單位**：親爸、親媽、**祖父母（內 + 外合為一個單位）**、其他，各占分母 1。
4. **D / R 計分**：要求 5 題（`w2s201`–`w2s206`）、回應 5 題（`w2s212`–`w2s219`），
   依主要照顧者的勾選加總，特殊碼 99 當 0。祖父母單位分數 = 有「涉入」之祖父母的平均
   （涉入 = 10 題中至少 1 題被勾選）；兩位都沒涉入則記 0、仍占分母 1。
5. **中位數切分** D、R（嚴格大於為「高」）→ 四類型：開明權威 / 專制權威 / 寬鬆放任 /
   忽視冷漠。

## 腳本與報告

報告 `.Rmd` 以敘述為主，完整程式碼集中於報告末的附錄；報告於設定階段 `source()` 對應
腳本後再計算正文數字，因此報告與腳本必然一致。

| 檔案 | 內容 |
|---|---|
| `scripts/rebuild_caregiver.R` | 步驟 1–3：品質過濾、聯集推主要照顧者 |
| `scripts/dr_scoring.R` | 步驟 4–5：D / R 計分、四類型 |
| `scripts/visualize.R` | 描述性視覺化（6 張圖） |
| `scripts/regression.R` | 連續迴歸、分層、ANOVA + Tukey、卡方、forward stepwise |
| `reports/照顧者重建與DR計分.{Rmd,html}` | 資料建置 + 描述性結果 |
| `reports/迴歸分析.{Rmd,html}` | 迴歸 / ANOVA / 卡方 |
| `outputs/figures/` | 圖檔（PNG） |
| `outputs/regression/` | 迴歸 / ANOVA / 卡方的彙總報表（CSV） |

執行順序：`rebuild_caregiver.R` → `dr_scoring.R` → `visualize.R` → `regression.R`
（或直接渲染報告，會自動依序 `source()`）。原始資料讀自 repo 根目錄的
`Data/stata/w2_j_s_v6.0.dta`。

## 衡量限制

1. **時間落差**：照顧者題問「國小畢業前」，教養 10 題反映較接近國中現況；兩題組對
   「祖父母是否涉入」約 76% 一致。
2. **內 / 外祖父母無法在計分中區分**：教養題的祖父 / 祖母欄只有性別軸、無內外軸；
   77 名「內外祖父母都在主要照顧者中」的學生併為一個祖父母單位。
3. **涉入檢核僅對祖父母施行**：補償問卷對祖父母只有「夫妻對」解析度（對父母為逐人辨識）。
4. **特殊碼 99 在加總時當 0**（每題約 0.5%，造成輕微低估）。
5. **中位數切分的機械效應**：離散分數加上「嚴格大於」規則，會把踩線者全歸低組。
